## 3.个人代码
* baseline1：第一版本，最初的特征，没有模型融合等；
* baseline2：第二版，加了很多特征，上分明显，上了30+名；
* baseline_CV：新版CV，相当于一种模型融合方式吧，2个万分点大概；
* baseline_pca：尝试PCA添加特征，线下有提升，奈何，线上没机会尝试；
* baseline_nn:神经网络，简单全连接，最后两天才写出来，比lgb低5个千；
